## Phương thức `unshift()`

### Cú pháp:

```js
array.unshift(item1, item2, ..., itemN);
```

### Mô tả:

- Phương thức `unshift()` được sử dụng để thêm một hoặc nhiều phần tử vào đầu mảng.

### Ví dụ:

```js
const numbers = [3, 4, 5];
numbers.unshift(1, 2);
console.log(numbers); // Output: [1, 2, 3, 4, 5]
```
